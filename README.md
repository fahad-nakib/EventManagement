# Celebration Station - Event Organizing System

**Celebration Station** is a comprehensive web-based platform designed to facilitate the organization and management of events like weddings, birthdays, and engagements. Developed as an academic project for **Green University of Bangladesh**, it provides a centralized solution to simplify complex event coordination.

---

## Technologies Used

The project follows a multi-tiered system architecture:



* **Frontend:** HTML, CSS, JavaScript, and Bootstrap for responsive design.
* **Backend:** PHP for server-side processing and data handling.
* **Database:** MySQL for relational data management.

---

## Key Features

### Guest Visitors
* **Event Browsing:** View a list of all upcoming events.
* **Detailed View:** Access specific information regarding event descriptions and logistics.
* **Contact Access:** Reach out to organizers via provided contact information.

### Registered Users
* **Secure Authentication:** Create accounts and log in to access personalized features.
* **Event Booking:** Book various celebrations such as weddings, engagements, and birthdays.
* **Profile Management:** View booked event history and customize profile details.

### Admin Interface
* **Event Control:** View, add, and manage details for all upcoming events.
* **Analytics:** Monitor user engagement and participation trends.

---

## Database Structure

The system manages data through three core tables:
1. **Users:** Stores names, emails, and encrypted passwords.
2. **Events:** Contains names, dates, times, locations, and descriptions.
3. **Admins:** Credentials for administrative platform access.

---

## Security Measures
* **Authentication:** Secure processes to protect user account integrity.
* **Data Validation:** Server-side checks to prevent SQL injection and cross-site scripting (XSS).
